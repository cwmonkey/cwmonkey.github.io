Usage and description

Drag the link to your shortcuts bar:

Clean Stream

When you are on a page with a stream (or any flash player) click the bookmarklet and it will remove everything that is not a flash player from the page. It will also stretch the flash player to fit the browser window.

This doesn't work 100% of the time, but if it messes the page up all you have to do is refresh.

Update - May 4, 2014: Works with HTML5 video. Hides text and removes spacing from tables. Fixed for Internet Explorer.

Update - Apr 3, 2014: Now works with the WWE Network player.

Update - Apr 4, 2014: Now works with https sites (like youtube). You will need to re-add the bookmarklet (and get rid of the old one) for this update to take effect.
