Usage and description

Install the Greasemonkey script below and take more control over your YouTube chat experience.

What This Script Does

  • Adds a right-click menu to chat authors which has options to change their visibility
  • Adds configurable sounds to mentions, keywords and users of your choice

What This Script Does NOT Do

  • This script does not and will not track you
  • This script will not inject any sort of ads or any other monetization content into the page
  • This script will not post anything or otherwise generate any content on your behalf
  • This script will not do anything to violate the YouTube terms of service

Versions

  • 0.3 (current) - 2023-10-17
    • Added keywords

Planned Features

  • Anti-keywords (mute/hide messages with certain words)
  • "Hard" mute (hide messages) - allow mods/owners to be seen?
  • "Hard" quiet (hide messages) - allow mods/owners to be seen?
  • Fav messages
  • Global settings (vs channel)
  • Global mute sound
  • User notes
  • User defined sounds
  • User specific sounds
